1 - Escrevendo APP-FEATURES.md

Você é uma persona do departamento comercial de uma empresa de desenvolvimento de software em JAVASCRIPT e acabou de discutir com o cliente quais os módulos que você precisa desenvolver com o teu time técnico. Escreva no arquivo APP-FEATURES.md, uma documentação clara que você passará para o PRODUCT MANAGER, que por sua vez vai redistribuir para o time de FRONT-END. O APP que precisamos em questão é um CADASTRO DE CLIENTES DE UMA OFICINA, CONTAS A PAGAR e MEU PERFIL. Iremos utilizar o VITE como já instalado neste projeto e o HEROUI para componentes REACT junto com TAILWIND.

2 - Criando a página de LOGIN

Crie uma tela de login no indexPage para a nossa aplicação. Vamos usar o campo email, senha, lembrar-me e o botão Login.

Lembre-se daqui pra frente sempre usando o HEROUI de componentes e o HTML com TAILWIND para estrutura.

3 - Como criar a estrutura do Layout usando Outlet do REACT ROUTER DOM.

Preciso que você crie a minha rota dashboard de minha aplicação. siga as instruções abaixo:

a - Header com 80px de largura. Lado esquerdo do header coloque uma logo quaquer relacionada a oficianas, pode usar um icone do Lucide-React se necessário. E no lado direito do header coloque um Avatar, mais a frase Seja bem-vindo e um nome qual embaixo do Seja bem-vindo e por ultimo um icone de logout.
b - Adicione uma sidebar do lado esquerdo com 3 menus: Dashboard, Clientes, Meu Perfil. Faça os botões com uma animação quando passar o mouse por cima qualquer.
c - Organize o layout do react-router-dom para usar o componente Outlet e a gente utilizar NESTED ROUTES para navegação entre as rotas, pois devemos sempre manter o HEADER e a SIDEBAR do MENU.
d - Já crie as páginas relacionadas ao DASHBOARD, MEU PERFIL e CLIENTES, mas não precisa colocar o conteudo dentro dela... apenas uma div escrito um TITULO de cada rota somente.

4 - Configure o react hook form neste componente... e e em cada componente do formulário utilize o controller do hook form.
